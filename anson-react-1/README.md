# Day 01: React Basics

This project focuses on the fundamental structure of a React application.

## Key Learnings

- Project initialization with Vite.
- Understanding the root component (`App.jsx`) and the entry point (`main.jsx`).
- Modern JSX syntax vs older `createElement` methods.
- Basic component composition (nesting components).

## Project Structure

- `src/components/UserProfile.jsx`: A container component for user information.
- `src/components/UserName.jsx`: Displays basic name details.
- `src/components/UserFavorite.jsx`: Displays user hobbies/favorites.
